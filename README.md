# 清风AI探索博客

这是一个基于Hugo框架的个人博客，主题为Stack，专注于分享AI技术与应用的知识。

## 特点

- 响应式设计
- 多语言支持
- 黑暗模式支持
- 搜索功能
- 文章分类与标签

## 本地开发

### 安装Hugo

```bash
# 安装Hugo扩展版
# Windows用户可以从GitHub下载并安装
```

### 克隆仓库

```bash
git clone https://github.com/yangzai-blog/yangzai-blog.git
cd yangzai-blog
git submodule update --init --recursive
```

### 本地运行

```bash
hugo server -D
```

浏览器访问 http://localhost:1313/ 即可查看博客。
